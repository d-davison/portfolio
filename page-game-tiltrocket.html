Tilt Rocket

Overview

(Personal Project - 2020)

Tilt Rocket is a side project I developed for mobile in Unity. Originally just a prototype to test the gyroscope of the mobile phone, this project turned into a full hyper-casual mobile game with unlockable rockets, challenges, and 25+ levels for the players to defeat.

When I originally released Tilt Rocket, the game, at it's bare bones was 1 rocket, and only 10 levels, with no customization or repeatability. I pushed the project out to publishers for some feedback. I got some great feedback, and publishers really liked the fast paced levels and the visuals, but thought the game could do with some more depth, repeatability and some brightening of the visuals. with this feedback, I added some extra features and made some changes to the core of the game. Some of the news features and improvements that I made:

  • New rockets for the user to customize and unlock
  • A new section, with more props, and dynamic visuals
  • A bump on the post processing to help the bright colours 'pop'
  • A focus on the baking of the level's textures to keep detail but minimize app size
  • Custom challenges for the user to complete for rewards
  • A rework on the monetization of the adverts
  • An option to use a joystick instead of the gyroscope for controls


Social Media

After I released the project to the Google Play Store, I use social medias such as instagram to raise awareness of the game, hopefully to push some organic downloads for the game rather than to fuel downloads through advertisement. I posted numerous times especially on Instagram, amassing over 400 followers organically through the content for this game and through others that I had developed.

Modelling

Although not my main focus, I do still really enjoy creating 3D assets, so for this project I also modelled all the 3D Assets. I created the 3D assets using Blender, mainly because it's the 3D software that I am most fluent in, but also because it's licensing would allow me to use it for free.
