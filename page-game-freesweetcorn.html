Free Sweetcorn

Content

Free Sweetcorn is a project that has been through a few iterations. It's initial concept came about when I was in college, and since then it has been developed and re-developed a few times; to investigate and show my growing skill-set, but mostly because it's a silly game with a silly concept and I like that.

For this game I really wanted to focus on the visuals and the gameplay of the app; experimenting with some more eciting visuals, and using 2D lighting to make a more dynamic and engaging environment. I used pulsing point lights to create glowing candlebera, and a glowing torch that the player holds to light the way.

For the gameplay, I developed the control system, using some dyanmic controls that matched their user's input position
